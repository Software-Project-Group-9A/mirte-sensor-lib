Class: IntervalPublisher

IntervalPublisher(ros, topicName, hz)

Interface-like class that can be extended by sensors that need their messages to be published at regular intervals. Usage requires provision of standard frequency for class in constructor and implementation of createSnapshot function.

Constructor

new IntervalPublisher(ros, topicName, hz)

Creates a new sensor publisher that publishes to the provided topic with a Regular interval.
Parameters:
Name Type Default Description
ros ROSLIB.Ros a ROS instance to publish to
topicName String name for the topic from ROS on which to publish data to
hz Number 10 a standard frequency for this type of object.
