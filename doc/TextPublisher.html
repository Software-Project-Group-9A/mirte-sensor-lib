Class: TextPublisher

TextPublisher(ros, topicName, inputElement, optionsopt)

TextPublisher publishes the text of an HTML input element. By default it publishes data whenever the enter key is pressed, but it can be configured sends the data every keypress. The data resulting from the text interactions is published as a ROS std_msgs/String message.
