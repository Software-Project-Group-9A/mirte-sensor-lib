Class: AmbientLightPublisher

AmbientLightPublisher(ros, topicName, hz)

AmbientLightPublisher publishes the amount of lux the camera receives By default it publishes data at the interval rate from parrent class IntervalPublisher VERY IMPORTANT: This feature is not fully suported over all browsers To enable in Chrome, go to: chrome://flags/ There enable: "Generic Sensor Extra Classes" The data resulting from the interactions is published as a ROS std_msgs/Int32 message.
