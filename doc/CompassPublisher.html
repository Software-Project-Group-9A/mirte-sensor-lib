Class: CompassPublisher

CompassPublisher(ros, topicName, hz)

CompassPublisher publishes the rotation as a compass By default it publishes data at the interval rate from parrent class IntervalPublisher The data resulting from the interactions is published as a ROS std_msgs/Int32 message.
